Details / Summary

How do I customize the theme?

Override the CSS custom properties in your own stylesheet. All colors, spacing, and typography can be changed through variables defined on :root.

What fonts are used?

Geist for headings and UI labels. Geist Mono for body text, code, and form inputs — giving everything a terminal feel.

Is it really classless?

For base elements, yes. Semantic HTML elements like headings, paragraphs, tables, forms, and code blocks are styled without any classes. A few optional utility classes are provided for buttons, alerts, and cards.
