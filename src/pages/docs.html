Install

Install wrc globally via npm:

npm install -g wrc

Or use it directly with npx:

npx wrc init my-project

Verify your installation:

$ wrc --version
wrc v2.4.1 (node v20.11.0)

Quick Start

Create a new project in seconds:

$ wrc init my-app
  Scaffolding project in ./my-app ...

  Done. Now run:
    cd my-app
    npm install
    npm run dev

The init command creates a directory with:

  • src/ — source files
  • public/ — static assets
  • wrc.config.js — project configuration
  • package.json — dependencies and scripts

Commands

wrc init <name>

Scaffold a new project from a template.

Flag Description Default
--template, -t Template to use default
--force, -f Overwrite existing directory false
--git Initialize git repository true
--install Run npm install after scaffold true

wrc dev

Start the development server with hot reload.

$ wrc dev --port 8080
  Server running at http://localhost:8080
  Watching for changes...

wrc build

Build for production. Outputs to dist/ by default.

$ wrc build
  Building for production...
  dist/index.html     1.2 kB
  dist/style.css      3.1 kB
  dist/main.js       12.4 kB

  Build complete in 340ms.

wrc add <plugin>

Add a plugin to the current project.

$ wrc add @wrc/tailwind
  Installing @wrc/tailwind...
  Updated wrc.config.js
  Done.

Configuration

All options live in wrc.config.js at the project root:

// wrc.config.js
export default {
  root: './src',
  outDir: './dist',
  port: 3000,
  open: true,
  plugins: [],
  build: {
    minify: true,
    sourcemap: false,
    target: 'es2022'
  }
}

Options

root Source directory for entry files "./src"
outDir Output directory for builds "./dist"
port Dev server port 3000
open Open browser on dev start true
build.minify Minify output files true
build.sourcemap Generate source maps false

Templates

Built-in templates for common project types:

Name Description Includes
default Vanilla HTML/CSS/JS None
react React with JSX React 19, ReactDOM
vue Vue 3 SFC Vue 3, Volar types
node Node.js API Express, nodemon
static Static site Markdown, layouts

Use any template with the --template flag:

wrc init my-app --template react

Plugins

Extend wrc with community and official plugins.

@wrc/tailwind — Tailwind CSS integration

Adds Tailwind CSS processing to the build pipeline. Auto-generates tailwind.config.js and PostCSS config.

wrc add @wrc/tailwind
@wrc/mdx — MDX support

Write pages in MDX with full JSX component support. Great for documentation sites and blogs.

wrc add @wrc/mdx
@wrc/icons — Icon bundling

Tree-shakeable icon imports from popular icon sets. Only bundles icons you actually use.

wrc add @wrc/icons
