Changelog

All notable changes to TerminalX CSS are documented here.

v0.3.0

2026-03-25 latest
  • added New .card component with rounded corners and surface background
  • added Alert variants: .alert-success, .alert-accent, .alert-error
  • added Progress bar styling with accent fill and smooth radius
  • fixed Navigation links now properly show active state via .active or aria-current
  • changed Default border radius increased from 4px to 6px for softer appearance

v0.2.0

2026-03-18
  • added Full form styling: text inputs, selects, textareas, checkboxes, radio buttons
  • added Button variants: .btn-ghost, .btn-error, .btn-block
  • added <details> / <summary> styling with accent hover and border animation
  • breaking Renamed --primary to --accent for consistency with design tokens
  • fixed Table header text now uppercase with proper letter spacing
  • fixed <kbd> elements no longer inherit monospace background in <pre> blocks

v0.1.0

2026-03-10
  • added Initial release with base reset and typography
  • added Geist + Geist Mono font integration via Google Fonts
  • added Dark theme with Factory.ai-inspired color palette
  • added CSS custom properties for full theme customization
  • added Classless styling for headings, paragraphs, links, lists, tables, code, blockquotes, and navigation
  • added Responsive breakpoint at 680px for mobile-friendly layout
