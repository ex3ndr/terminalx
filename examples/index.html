TerminalX CSS

A minimal CSS library for building terminal-style UIs. Dark theme, monospace body, soft details. Just link the stylesheet and write HTML.

Get Started

Copy terminalx.css into your project and add one line.

<link rel="stylesheet" href="terminalx.css">

No build tools. No dependencies. No classes needed for base elements.

Aa

Geist Fonts

Geist Mono for body text gives the terminal feel. Geist Sans for headings creates clear hierarchy.

{ }

Classless Core

Semantic HTML elements are styled by default. Headings, tables, forms, code blocks — all look good out of the box.

--

CSS Variables

Every color, spacing value, and font is a custom property. Override --accent to retheme the entire library.

●

Soft Dark Theme

Not harsh black. Subtle surface layers from #0c0c0c to #222 with orange accents inspired by Factory.ai.

▢

Components

Modals, toasts, loaders, badges, toggles, cards, alerts, pills, and more — all optional utility classes.

↔

Responsive

Single breakpoint at 680px. Sidebar layouts collapse, nav stacks, tables shrink. Works on any screen.

Customization

Override any variable to match your brand:

:root {
  --accent: #8b5cf6;       /* purple instead of orange */
  --accent-hover: #a78bfa;
  --accent-muted: rgba(139, 92, 246, 0.15);
  --bg: #111111;           /* slightly lighter */
  --radius: 8px;           /* rounder corners */
}
